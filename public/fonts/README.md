# Gotham font (local)

This project references Gotham as the global font.

Due to licensing, Gotham is not included.

## How to enable

Place your own licensed Gotham webfont files in this folder:

- `Gotham-Regular.woff2` (weight 400)
- `Gotham-Medium.woff2` (weight 500)
- `Gotham-Bold.woff2` (weight 700)

If the files are missing, the overlay falls back to system fonts.
